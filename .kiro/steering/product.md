# Product Overview

Plataforma Educativa de Python para Análisis Financiero - An interactive learning platform for teaching Python programming with a focus on financial analysis to Financial Engineering students.

## Core Purpose

Educational web platform designed for the "Lenguajes de Programación" course at ITM, targeting students with no prior programming experience. The platform provides an interactive, gamified learning experience similar to Codecademy/DataCamp, specialized in financial analysis using Python.

## Key Features

- Interactive 4-panel laboratory interface (Content, Code Editor, Results, Exercises)
- In-browser Python execution using Pyodide (no server-side execution needed)
- Real-time financial data integration via yfinance (banking sector focus)
- Side-by-side Python/R code comparison for students familiar with R
- Gamification: badges, XP points, progress bars, daily challenges, leaderboards
- Team-based project submissions with GitHub integration
- Institutional authentication (@correo.itm.edu.co)
- Professor dashboard for monitoring student progress and grading
- Structured evaluation: 5 components (Trabajo_1 20%, Trabajo_2 20%, Concurso 20%, Examen Final 20%, Seguimiento 20%)

## Target Users

- Students: Financial Engineering students learning Python for the first time
- Professors: Course instructors managing ~30-50 students per semester
- Class schedule: Tuesdays and Thursdays 6:00 PM - 8:00 PM, ~34 classes total

## Technical Constraints

- Must use free tier cloud services exclusively
- Support 30-50 concurrent students per semester
- Classes conducted via Microsoft Teams screen sharing
- Integration with GitHub for version control and Google Colab for notebooks
